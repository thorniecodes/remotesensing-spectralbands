# About

This guide started as a personal reference tool — somewhere quick to look up spectral bands, wavelength ranges, and index applicability without digging through sensor documentation every time.

It was built to support geospatial modeling and research, and shared publicly in case it's useful to others. It is not comprehensive. The band data and wavelength specifications were gathered from publicly available agency documentation; full citations are on the [Sources & References]({{ '/sources' | relative_url }}) page.
